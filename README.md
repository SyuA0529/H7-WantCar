## ❤️‍🔥 프로젝트 소개

현대자동차그룹 소프티어부트캠프 2기에서 진행한 내 차 만들기 플로우 개선 프로젝트입니다. <br/>
Naughty팀의 기획과 디자인을 바탕으로 Front-End와 Andriod의 화면 개발을 진행했습니다. <br/>

**'내가 살 차, 이제 감이 아닌 데이터로.'** 라는 키워드로 현대만의 실질적인 데이터를 중점으로 한 서비스입니다.
